## Requirements for installing git-filter-repo
- Python v3.x
- Python pip

## Step to install
```bash
python -m pip install git-filter-repo
```

## Usage

```bash
git filter-repo --path FOLDER-NAME/
```

## Troubleshooting
### pip is missing
```bash
python -m pip install --upgrade pip
```
