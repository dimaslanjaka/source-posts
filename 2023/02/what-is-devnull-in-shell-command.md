| argument | argument description |
| :--- | :--- |
|   `> /dev/null`  | throw away **stdout** |
|   `1> /dev/null` | throw away **stdout** |
|   `2> /dev/null` | throw away **stderr** |
|   `&> /dev/null` | throw away both **stdout** *and* **stderr** |
